沿回头路拾贝

错题整理，知识点提取，原错题不一定放出。

  1. *用于解包，可用于列表。

  2. 用enumerate()来替代range()，枚举函数，返回（return）：迭代器

  3. 返回字典中，访问键值，考虑不存在情况，e.g:

    key = grade.get('物理-key',0)

    键不存在返回第二个参数，省略的话，未找到返回None

  4. 逆置函数→列表，reverse()

  5. 匿名函数lambda() 省略return，一行解决问题

  6. yield生成器/迭代器，为了避免使用列表（list），节省内存

    生成器generator

    迭代器iterator

    ​ iter()迭代函数可作用上方两个

  7. 装饰器 使用@附加在任意一函数定义前,后跟定义的函数作为装饰器函数的参数，举个栗子：

  8. 断言assert，调试，声明表达式的布尔值，必须为真的判定，也可以作为输入参数格式、类型验证

  9. 复制列表 copy list=list[:]

  10. 集合set 列表推导式/集合推导式 有以下方法：

    • set.add()
    • set.update()
    • set.remove() 无出错
    • set.discard() 无不出错 移除指定的集合元素
    • set.pop() 随机删除 return删除的元素
    • set.clear() 清空

  11. 异常捕获的方法

  12. 传递参数，带*参数以元祖(tuple)形式导入，存放所有未命名的内置函数

  13. map函数，提供的序列作映射，返回迭代器，内置函数，（函数，序列）类比一下Java的哈希

  14. 闭包，外函数里定义内函数，外函数返回内函数中值，内函数中对使用外函数中值需要加个 nolocal(理解成改变变量作用域)：nolocal vs global

  15. zip()函数，以一个或多个序列作为参数，返回一个元祖的列表

  16. 排序，不用sort，递归，用min，每次remove掉最小，继续（这一题我也不知道自己记了个啥）

  17. 创建字典的方法 dict()

  18. 魔法方法有：__init__、__new__、__del__、__str__,print(return)

  19. sort与sorted区别，后者有返回，sort直接改变序列

  20. 八进制oct()， 十六进制hex()
